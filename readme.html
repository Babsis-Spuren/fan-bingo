Anleitung
Installation
iPhone · Safari
Zum Home-Bildschirm hinzufügen
  1. Link in Safari öffnen (nicht Chrome oder Firefox)
  2. Das Teilen-Symbol antippen (Kasten mit Pfeil nach oben)
  3. „Zum Home-Bildschirm" auswählen
  4. „Hinzufügen" bestätigen

Die App startet danach vollbild – ohne Browser-Leiste, wie eine echte App.

Android · Chrome
App installieren
  1. Link in Chrome öffnen
  2. Banner unten antippen: „App installieren"
  3. Oder: Drei-Punkte-Menü → „Zum Startbildschirm hinzufügen"

Ohne Installation: Die App funktioniert auch einfach im Browser. Karten bleiben gespeichert, solange der Browser-Cache nicht geleert wird.

So funktioniert's

Bingo spielen

  1. JSON-Datei mit Kategorien importieren – auf „Importieren" tippen und die Datei auswählen
  2. Auf „Karte generieren" tippen – die App mischt automatisch eine zufällige Karte
  3. Wird ein Song gespielt? Feld antippen – es leuchtet auf
  4. Ist eine Reihe voll (waagerecht, senkrecht oder diagonal)? BINGO!

Karten verwalten

  • Bis zu 3 Karten können gleichzeitig gespeichert werden
  • Zwischen Karten wechseln: oben auf den Slot tippen
  • Karte löschen: kleines × im Slot antippen
  • Zurücksetzen löscht alle Markierungen, behält aber die Karte
  • Neu mischen erstellt eine komplett neue Zufallskarte
Features
  • Eigene Kategorien und Songs per JSON-Datei importierbar
  • 3 speicherbare Karten – bleiben auch nach App-Schließen erhalten
  • BINGO-Erkennung: Zeilen, Spalten und Diagonalen
  • Offline-fähig nach erstem Laden
  • Kein Login · Keine Werbung · Keine Daten werden gesendet

Installation
iPhone · Safari
Add to Home Screen
  1. Open the link in Safari (not Chrome or Firefox)
  2. Tap the Share icon (box with arrow pointing up)
  3. Select "Add to Home Screen"
  4. Confirm with "Add"

The app then launches fullscreen – no browser bar, just like a native app.

Android · Chrome
Install the app
  1. Open the link in Chrome
  2. Tap the banner at the bottom: "Install app"
  3. Or: three-dot menu → "Add to Home Screen"

Without installing: The app also works directly in the browser. Cards are saved as long as the browser cache isn't cleared.

How it works

Playing bingo

  1. Import a JSON file with categories – tap "Import" and select the file
  2. Tap "Generate card" – the app randomly creates a unique card
  3. Song being played? Tap the cell – it lights up
  4. Full row (horizontal, vertical or diagonal)? BINGO!

Managing cards

  • Up to 3 cards can be saved at the same time
  • Switch between cards: tap the slot at the top
  • Delete a card: tap the small × in the slot
  • Reset clears all marks but keeps the card
  • Reshuffle generates a completely new random card
Features
  • Import your own categories and songs via JSON file
  • 3 saved cards – persisted even after closing the app
  • BINGO detection: rows, columns and diagonals
  • Works offline after first load
  • No login · No ads · No data is transmitted
