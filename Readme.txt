Readme:
So benutzt du die Seite

iPhone:

Link in Safari öffnen (nicht Chrome!)
Teilen-Symbol → „Zum Home-Bildschirm"
App startet wie eine native App, vollbild, kein Browser-UI

Android:

Link in Chrome öffnen
Banner erscheint automatisch: „App installieren"
Oder: Drei-Punkte-Menü → „Zum Startbildschirm hinzufügen"

Ohne Installation: App funktioniert auch einfach im Browser,
Karten bleiben gespeichert solange der Browser-Cache nicht geleert wird.

Features

131 Songs aus 9 Kategorien, zufällig gemischt
3 speicherbare Karten (bleiben nach App-Schließen erhalten)
BINGO-Erkennung (Zeilen, Spalten, Diagonalen)
Offline-fähig nach erstem Laden
Kein Login, keine Daten werden gesendet
